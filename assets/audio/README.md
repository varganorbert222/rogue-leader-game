# Audio export guidelines

| Type | Format | Sample rate | Notes |
|------|--------|-------------|--------|
| Music | MP3 or OGG | 44.1 kHz | Seamless loop |
| SFX | MP3 or OGG | 44.1 kHz | Trim silence, keep under ~2s |

Paths are registered in `audio/manifest.json`. Keep manifest **ids** stable; change filenames only if you update the manifest.
